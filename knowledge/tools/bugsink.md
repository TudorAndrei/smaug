Self-hosted error tracking solution that provides a drop-in Sentry alternative. Bugsink works seamlessly with the Sentry SDK by simply changing the DSN URL, making it easy for teams to migrate away from Sentry while keeping their existing SDK integration.

## Key Features

- Works with existing Sentry SDK - just change the DSN URL
- Self-hosted error tracking
- Drop-in Sentry replacement
- Python-based implementation

## Why Bookmarked

Useful for teams looking to reduce subscription costs by moving off Sentry to a self-hosted solution. The fact that it works with Sentry SDK makes migration straightforward.
